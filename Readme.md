# DDtimer

N ゲージのイベント運転体験用タイマー <br>
スタッフ側/操縦者側で同じタイマーを見ることができる。<br>
外線/内線それぞれのタイマー設定ができる。<br>

## Processing のみシンプルバージョン

パソコン＋ iPad もしくはモニター<br>
残り時間を左右に分ける<br>
パソコンとモニタは左右別にする<br>

#### ボタン

- リセットボタン(1×2)
- スタート/ストップボタン(1×2)
- 数字＋－ボタン(2×2)
  計 8 個

## 記録等高度機能導入バージョン

各種記録を取るようにする<br>

- 番号と使用時間
- 時刻

## マイコン連携バージョン

ESP32 使用？<br>
残り時間を LED や 7 セグディスプレイ、LCD 等で表示<br>
Wi-Fi もしくは Bluetooth を使った無線<br>
74HC165 を使ってボタン複数個を 1 つの端子で使えそう<br>
