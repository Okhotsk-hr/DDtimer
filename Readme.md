# DDtimer

N ゲージのイベント運転体験用タイマー
スタッフ側/操縦者側で同じタイマーを見ることができる。
外線/内線それぞれのタイマー設定ができる。

## Processing のみシンプルバージョン

パソコン＋ iPad もしくはモニター
残り時間を左右に分ける
パソコンとモニタは左右別にする

#### ボタン

- リセットボタン(1×2)
- スタート/ストップボタン(1×2)
- 数字＋－ボタン(2×2)
  計 8 個

## 記録等高度機能導入バージョン

各種記録を取るようにする

- 番号と使用時間
- 時刻

## マイコン連携バージョン

ESP32 使用？
残り時間を LED や 7 セグディスプレイ、LCD 等で表示
Wi-Fi もしくは Bluetooth を使った無線
74HC165 を使ってボタン複数個を 1 つの端子で使えそう
